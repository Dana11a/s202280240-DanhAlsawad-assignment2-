# Assignment 2 – Personal Portfolio

##  Project Description

This project is an improved version of my personal portfolio website built using HTML, CSS, and JavaScript.
It extends Assignment 1 by adding interactive features, better user experience, and a modern UI design.

The website includes About, Projects, and Contact sections with smooth navigation, dynamic behavior, and user feedback.

---

## Features

- Responsive design (mobile-friendly)
- Dark / Light mode toggle with localStorage (remembers user preference)
- Smooth scrolling navigation between sections
- Active section highlighting in navbar
- Dynamic project filtering by category
- Contact form validation with user feedback messages
- Scroll reveal animations for sections
- Improved UI design with modern styling


---

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Media Queries, Animations)
- JavaScript (DOM Manipulation, Events, localStorage)

## How to Run Locally

1. Download or clone the repository.
2. Open index.html in your browser.
3. Scroll through the page to navigate between sections.
4. Use the navigation menu to jump to sections.
5. Toggle theme using the icon button.
6. Use project filter buttons to display specific categories.
7. Fill the contact form to test validation and feedback.
